# /today

## Purpose
Morning orientation. Suggest 3 priority focuses based on the current state of the wiki.

## Workflow
1. Read `wiki/hot.md` and the last 5 entries of `wiki/log.md`.
2. Read `wiki/index.md` to see what areas are thinly covered.
3. Suggest 3 priority focuses for the day, drawn from:
   - Open questions in `hot.md`
   - Sources sitting in `raw/` that have not been ingested yet
   - Concepts mentioned 3+ times but lacking their own page
   - Pages with `confidence: low` that need follow-up
4. Present as a short numbered list. Wait for the user to pick one.

## When to Use
At the start of a working session when you want a focus, not a briefing.
